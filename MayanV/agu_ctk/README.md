Awakatek bilingual corpus
==========================================================================================================

## About this corpus
This package contains an Awakatek and Chalchitek corpus extracted from the _Diccionario awakateko y chalchiteko_, published by the [Proyecto Lingüístico Francisco Marroquín]()https://plfm.org/ (PLFM).
