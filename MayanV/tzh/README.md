Tektiteko bilingual corpus
==========================================================================================================

## About this corpus
This package contains an Tzʼeltal corpus extracted from a crawl of the _[Diccionario Tseltal en línea](https://ditsel.aldelim.org/)_ published by [Gilles Polian](https://documentaciontseltal.aldelim.org/equipo/gilles.html).
